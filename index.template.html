Motion Guidance: Diffusion-Based Image Editing with Differentiable Motion Estimators

Daniel Geng, Andrew Owens
University of Michigan
Paper arXiv Video Code Data

We present Motion Guidance, a method to achieve motion-based editing. Given an image to edit and a flow field, indicating where each pixel should go, we produce a new image with the desired motion. See below for examples.
